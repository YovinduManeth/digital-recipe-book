# Digital Recipe Book 🍲

## 📌 Project Overview

The Digital Recipe Book is a web-based application developed for the ICT 1209 Web Technologies mini project.
It allows users to browse, search, and manage cooking recipes in an easy and organized way.

## 🎯 Purpose

The purpose of this project is to create a simple and user-friendly platform where users can access and manage different types of recipes online.

## 👥 Target Audience

* Students
* Home cooks
* Food enthusiasts
* Anyone interested in cooking

## ⚙️ Technology Stack

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* PHP (future phase)
* MySQL (future phase)
